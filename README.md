# 🤖 Мой Telegram Бот-Менеджер Проектов

Привет! 👋 Это мой проект — крутой телеграм-бот, который помогает ИТ-шникам, разработчикам и студентам навести полный порядок в своем портфолио. 

Больше не нужно хранить ссылки на свои работы в заметках! Бот аккуратно разложит всё по полочкам прямо в Telegram, используя базу данных SQLite. 📂

---

## ✨ Что умеет этот бот?

*   ➕ **Добавлять проекты** — за пару кликов сохраняет имя проекта и ссылку на него.
*   ⚙️ **Менять статусы** — можно отмечать, на каком этапе задача (например: «В разработке» или «Завершено»).
*   🛠️ **Добавлять технологии** — к каждому проекту можно привязать стек (Python, SQL и др.).
*   📝 **Обновлять инфо** — если что-то изменилось, можно легко отредактировать описание или ссылку.
*   🗑️ **Удалять** — ненужные или старые тест-проекты можно легко стереть.
*   🔍 **Быстрый поиск** — просто напиши боту название своего проекта, и он тут же выдаст красивую карточку со всей информацией!

---

## 🛠️ Из чего сделан проект?

*   **Язык:** Python 🐍
*   **Библиотека:** pyTelegramBotAPI (Telebot) 🤖
*   **База данных:** SQLite3 🗄️

---

## 🚀 Как запустить бота у себя?

### 1. Скачай проект к себе на компьютер 💻
```bash
git clone [https://github.com/ЗАМЕНИ_НА_СВОЙ_ЛОГИН/ЗАМЕНИ_НА_ИМЯ_РЕПОЗИТОРИЯ.git](https://github.com/ЗАМЕНИ_НА_СВОЙ_ЛОГИН/ЗАМЕНИ_НА_ИМЯ_РЕПОЗИТОРИЯ.git)

```

### 2. Установи нужную библиотеку 📦

```bash
pip install pytelegrambotapi

```

### 3. Настрой токен 🔑

Создай рядом с файлом `main.py` файл `config.py` и запиши туда свой токен от [@BotFather](https://t.me/BotFather):

```python
TOKEN = "ТВОЙ_ТЕЛЕГРАМ_ТОКЕН"
DATABASE = "port.db"

```

### 4. Включай и пользуйся! 🔥

```bash
python main.py

```

*База данных создастся сама при первом запуске!*

---

## 📋 Команды, которые есть в боте

* `/start` — Начать работу с ботом 🚀
* `/info` — Посмотреть подсказки по всем командам 📖
* `/new_project` — Добавить новый крутой проект ➕
* `/projects` — Посмотреть список всех твоих работ 🗂️
* `/skills` — Прокачать проект и добавить к нему технологии 🎒
* `/update_projects` — Быстро отредактировать данные 🔄
* `/delete` — Удалить проект из базы ❌
